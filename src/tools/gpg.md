# gpg

<https://www.gnupg.org/>

## Useful Commands

| Command                             | Notes                           |
| ----------------------------------- | ------------------------------- |
| `gpg --delete-key [uid]`            | Delete public key for `uid`     |
| `gpg --delete-secret-keys [key-id]` | Delete private key for `key-id` |
| `gpg --list-keys`                   | List public keys                |
| `gpg --list-secret-keys`            | List private keys               |
